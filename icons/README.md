# Theme Icons

This directory contains the icon files for the Claude Chrome Theme.

## Required Icons

The following icon sizes are needed:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Icon Design

The icons should feature the Claude brand colors:
- Primary: #C15F3C (Crail - terracotta/orange-brown)
- Background: #1a1816 (dark warm brown)
- Accent: #F4F3EE (warm off-white)

## Creating Icons

You can create these icons using any image editor. The icons should be simple and recognizable, ideally featuring:
- Claude branding elements
- The warm terracotta color scheme
- Clear visibility at small sizes

## Temporary Solution

While the theme will load without icons, Chrome will show a default icon. For testing purposes, you can:
1. Use placeholder icons from an existing Chrome theme
2. Create simple solid-color icons using an image editor
3. Generate icons using online tools like favicon.io or similar services
